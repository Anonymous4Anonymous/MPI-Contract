VSL Dependencies version 1.7

======================= Overview =======================

This distribution contains all the dependencies for ABC/CIVL/SARL and is organized as follows.

.\lib contains the following libraries:

ANTLR v3.5.2 and v4.4
Hamcrest-core v1.3
Junit v4.11
PerfAnal

.\licenses contains the licenses for each library in .\lib.

.\src contains java source code for some of the libraries in .\lib and
is included here for debugging purposes.
Besides, the jacoco (v0.7.5) source distribution is included here, too.


======================= Change from version 1.6 ========================
Updated jacoco to the latest (v0.7.5).
